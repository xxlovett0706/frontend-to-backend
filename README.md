# frontend-to-backend

## 如何快速学习一个新技术或者知识点

- 我们有一个需求
    1. 项目开发时，我们需要解决某个问题
    2. 跳槽，学习新技术 
- 以页面局部刷新问题为例
    1. 先看看使用现有的技术能否解决这个问题（iframe）
    2. 发现使用传统方法可以解决，但是不够好，使用新技术（ajax）
- 学习新技术或者知识点
    1. 原理
    2. 基本语法
- 快速入门案例（这里不涉及技术细节）
    1. 简单入门
    2. 了解新技术或知识点的基本使用
- **讨论这个新技术或知识点的细节（体现一个程序员水平）**
    1. 怎么使用规范
    2. 使用陷阱
    3. 需要使用时，注意什么
